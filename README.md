# Mika's Woodcutter Extension: Stonecutter parity for wood blocks
A datapack that extends Minecraft's stonecutter to accept wood blocks, facilitating wood item crafting.
Wood items can be obtained both using logs as well as planks.

## Wood Variants
- Oak *unfinished*
- Spruce *unfinished*
- Birch *finished*
- Jungle *finished*
- Acacia *finished*
- Dark Oak *finished*
- Mangrove *finished*
- Cherry *not yet implemented*
- Pale Oak *unfinished*
- Crimson *not yet implemented*
- Warped *not yet implemented*
